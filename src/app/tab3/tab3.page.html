Kolmas tab (Info) Tab 3

Tere!

Esimesel tabil saab sisestada, kuvada, uuendada ja kustutada andmeid. Lisain ainult "Nime" sinna millele tuleb siis automaatselt oma id.

Teisel tabil saab teha kaameraga pilti ja salvestab selle. Kui peale vajutada pildile, siis on võimalik kustutada pilt.

Kolmas tab on see siin mida hetkel loed, siin on lihtsalt suvaline info. Alguses ma proovisin teha vähe keerulisemat 'äppi', aga see ei tulnud nii hästi välja. Vahepeal tuli Ionicul uuendus ning nüüd on vaja Java JDK ja Android Studiot, et saaks 'äppi' testida telefonis.

SQL info asub: tab1.page.ts failis ja tab1.page.html on html osa sellele.
